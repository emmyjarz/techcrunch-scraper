# Techcrunch-scraper
A web app that lets users view and leave comments on the latest news which is scraped from another site by Cheerio. Deploy on [heroku](https://emmyscraper.herokuapp.com/) 

## Full-Stack site
- Bootstrap3
- Handlebars
- Cheerio
- Morgan
- Express
- Node.js
- MongoDB/Mongoose
